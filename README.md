# AI Agriculture Assistant 2026

An AI-powered agriculture assistant for crop disease detection and soil analysis using machine learning.

## Features
- **Crop Disease Detection**: Real-time leaf scan to detect diseases and provide treatment suggestions.
- **Soil Analysis**: Analyze soil characteristics from images or entered parameters (Nitrogen, Phosphorus, Potassium, pH).
- **Offline ML Support**: Optimized TensorFlow Lite models running directly on device.
- **Multi-language Support**: Available in English, Hindi, Kannada, Marathi, Tamil, and Telugu.

## Tech Stack
- **Android Studio**: Kotlin-based development.
- **TensorFlow Lite**: On-device machine learning inference.
- **CNN (Convolutional Neural Networks)**: High-accuracy models for plant and soil classification.
- **Material Design**: Premium and responsive user interface.

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/DARSHAN-A17/AI-Agriculture-Assistant-2026.git
   ```
2. Open the project in **Android Studio** (Hedgehog or later).
3. Wait for Gradle Sync to complete.
4. Build and Run on an Android device (API 24+).

## Usage
1. Open the app and Register/Login.
2. From the Dashboard, select a module:
   - **Plant Disease**: Capture or upload a leaf image to detect diseases.
   - **Soil Analysis**: Enter soil parameters or analyze soil images.
3. View results and optional treatment/improvement suggestions.
4. Save reports to your History for future reference.
